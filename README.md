# Triskcraft SMP

Plugins, herramientas y servicios para el servidor de Minecraft **Triskcraft SMP**.

## Sobre el proyecto

Triskcraft SMP es una comunidad de Minecraft enfocada en crear experiencias únicas mediante
plugins personalizados, automatización del servidor y herramientas web.

Nuestro objetivo es construir un ecosistema completo que incluya:

- Plugins personalizados
- Integraciones con Discord
- Panel web para gestión
- Sistemas de estadísticas y actividad

---

## Proyectos principales

| Proyecto | Descripción |
|--------|-------------|
| ChatBridge | Conecta el chat de Minecraft con Discord |
| bot-inactividad | Sistema para medir inactividad de jugadores |
| triskcraft-web | Sitio web de la comunidad |
| CubeHelper | Herramientas para administración |
